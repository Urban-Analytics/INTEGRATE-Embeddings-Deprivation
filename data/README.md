# Data

The data files are too large to include in this repository (~6 GB) and must be obtained separately.

## Expected directory structure

```
data/
├── raw/
│   ├── points_with_embeddings.pkl          # Source point metadata with image paths
│   ├── imd/
│   │   ├── File_2_-_IoD2025_Domains_of_Deprivation.xlsx
│   │   └── File_2_-_IoD2019_Domains_of_Deprivation.xlsx
│   └── spatial/
│       └── LSOAs_2021/                     # LSOA 2021 shapefile set
│           ├── LSOA_2021_EW_BSC_V4.shp
│           ├── LSOA_2021_EW_BSC_V4.shx
│           ├── LSOA_2021_EW_BSC_V4.dbf
│           ├── LSOA_2021_EW_BSC_V4.prj
│           ├── LSOA_2021_EW_BSC_V4.cpg
│           └── LSOA_2021_EW_BSC_V4.shp.xml
├── processed/
│   ├── street_data.h5                      # HDF5 data store (images, embeddings, metadata)
│   ├── one_row_per_image_cleaned.pkl       # Output of notebook 3
│   ├── one_row_per_image_cleaned_with_cluster_numbers.pkl  # Output of notebook 5
│   └── per_lsoa_embedding_summaries/
│       ├── mean_embedding.pkl
│       ├── median_embedding.pkl
│       └── median_embedding_per_cluster.pkl
└── models/
    ├── best_model.joblib                   # Output of notebook 4
    └── model_testing/
        └── cluster_subsample_results.pkl   # Output of notebook 6
```

## Data sources

- **IMD (Index of Multiple Deprivation)**: Available from the [English Indices of Deprivation](https://www.gov.uk/government/statistics/english-indices-of-deprivation-2019) and the 2025 update.
- **LSOA boundaries**: Available from the [ONS Open Geography Portal](https://geoportal.statistics.gov.uk/).
- **Street view images and embeddings**: Generated by the upstream sampling script (notebook 1 creates the HDF5 store from source data).

## Notes

- The `processed/` and `models/` files are generated by running the notebooks in sequence. Only the `raw/` files are true external inputs.
- If you have the raw data, you can regenerate everything by running notebooks 1 through 8 in order.
